# 🚀 Quick Start - Admin Subscription Management

## 1️⃣ Access the Feature

1. **Login** as an admin user
2. Click **Admin Panel** (top right)
3. Click **Subscriptions** in the sidebar (👑 Crown icon)

---

## 2️⃣ Give Pro to a User

**For Free Users:**
```
1. Find the user in the list
2. Click "Actions"
3. Choose:
   • Give Pro (30 days)   - 1 month trial
   • Give Pro (90 days)   - 3 months
   • Give Pro (1 year)    - 12 months
4. Confirm
✅ Done! User now has Pro access
```

---

## 3️⃣ Extend Pro Subscription

**For Active Pro Users:**
```
1. Find the Pro user
2. Click "Actions"
3. Choose:
   • Extend +30 days   - Add 1 month
   • Extend +90 days   - Add 3 months
   • Extend +1 year    - Add 12 months
4. Confirm
✅ Done! Time added to their expiry date
```

**Example:**
- Current expiry: Dec 31, 2026
- Click "Extend +90 days"
- New expiry: Mar 31, 2027 ✨

---

## 4️⃣ Cancel Subscription

**Keeps Pro until expiry:**
```
1. Find the Pro user
2. Click "Actions"
3. Click "Cancel (keep until expiry)"
4. Confirm
✅ Done! User keeps Pro until expiry date
   Then automatically becomes Free
```

---

## 5️⃣ Revoke Pro Immediately

**Removes Pro access now:**
```
1. Find the Pro user
2. Click "Actions"
3. Click "Revoke (immediate)"
4. Confirm
✅ Done! User immediately loses Pro access
```

---

## 6️⃣ Reactivate Canceled Subscription

**For Canceled Pro Users:**
```
1. Find the canceled Pro user
2. Click "Actions"
3. Click "Reactivate +30 days"
4. Confirm
✅ Done! Subscription active again with 30 days added
```

---

## 🎯 Quick Decision Guide

**Need to give someone Pro?**
→ Use "Give Pro" (30/90/365 days)

**Need to extend existing Pro?**
→ Use "Extend" (+30/+90/+365 days)

**User wants to cancel but finish their period?**
→ Use "Cancel (keep until expiry)"

**Need to remove access immediately?**
→ Use "Revoke (immediate)"

**User changed their mind about canceling?**
→ Use "Reactivate +30 days"

## ⚠️ Important Notes

1. **All actions require confirmation** - No accidental changes
2. **All actions are logged** - Check Audit Log for history
3. **Extend adds to current expiry** - Not from today
4. **Cancel keeps access** - Until original expiry date
5. **Revoke is immediate** - User loses Pro right away

## 🆘 Troubleshooting

**Can't see the Subscriptions page?**
→ Make sure you're logged in as an admin

**Actions not working?**
→ Check your internet connection
→ Refresh the page and try again

**Need to undo an action?**
→ Use the opposite action:
  - Gave Pro by mistake? → Revoke
  - Revoked by mistake? → Give Pro again
  - Canceled by mistake? → Reactivate

**Want to see what happened?**
→ Go to Admin Panel → Audit Log
→ Search for subscription actions
